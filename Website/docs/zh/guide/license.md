3. **署名要求：** 必须署名上述所有贡献者

## 问题

关于许可证或使用权限的问题：

1. **代码许可：** 参考相应的 GPL 许可证条款
2. **艺术作品使用：** 联系原作者获取授权
3. **商业使用：** 查看 CC BY-NC-SA 4.0 限制条款
4. **分发：** 确保遵守所有适用的许可证

::: tip 合规提示
在重新分发或修改 SukiSU-Ultra 时，请确保遵守各个组件的所有适用许可证和署名要求。
:::
